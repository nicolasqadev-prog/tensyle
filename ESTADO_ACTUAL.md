## 🎬 VIDEOS (PRÓXIMO PASO)

### Cómo Generarlos:

**Opción 1: AUTOMÁTICO (Recomendado)**
1. Abre `generate_videos_local.html` en tu navegador
2. Haz clic en "Generar Video" en cada tarjeta
3. Descarga los GIF
4. Coloca en carpeta `./videos/`
5. Listo - aparecerán en la sección "Nuestro Trabajo"

**Opción 2: SIN VIDEOS**
- Las imágenes estáticas ya son suficientes prueba
- El efecto hover antes/después funciona sin videos
- Mantiene el sitio más ligero

## 🚀 PARA PONER EN PRODUCCIÓN

### Requisitos:

1. **Para la web estática**: Solo necesitas `index.html` y tus imágenes
2. **Para cotizaciones**: Necesitas ejecutar `python app.py`

### Comandos:

```bash
# Para ver la web
Abre index.html en tu navegador

# Para generar videos
Abre generate_videos_local.html en tu navegador

# Para que funcione el formulario de cotización
python app.py
# Luego abre http://localhost:5000
```
